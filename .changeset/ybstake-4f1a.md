YieldBasis: `YieldBasisLpFacet.setStakedMode()` and `DynamicYieldBasisLpFacet.setStakedMode()` now enforce collateral neutrality in both the stake and unstake branches when sweeping account gauge state. Each branch snapshots the shortfall baseline before the gauge call, reconciles tracked shares to actual recoverable LP, then re-checks collateral. The stake branch additionally rejects a lossy deposit outright (`require(gauge.convertToAssets(sharesMinted) >= lpSent, "Lossy stake")`). The unstake branch previously redeemed and reconciled with no baseline or enforce, so a gauge redeem returning fewer LP than `convertToAssets(shares)` implied could silently under-secure an at-limit position; it now reverts `UndercollateralizedDebt` when the redeem realizes new loss, while staying lenient to pre-existing undercollateralization (same-block delta check) so honest de-risking is never blocked. As a result, unstake now reads `pricePerShare`/`preview_withdraw` and reverts if the YB market is paused (symmetric with the stake branch). The `deposit()` auto-stake path is intentionally left unguarded so adding collateral to an already-drifted gauge stays deadlock-free. Solidity-only fix, ABI unchanged.
